Regrammed from <a href="https://www.instagram.com/alchemydigitalau">@alchemydigitalau</a>: Have decided to completely rebuild the cloud file-sharing system that I built earlier this year. Never quite got round to finishing what I had in mind, and time - and technology - have moved on. Got plans to focus the service and increase performance dramatically. In the meantime I've put the existing code base up on GitHub for anyone who's vaguely interested. No license - do what you like with it!
